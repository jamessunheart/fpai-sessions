Before doing ANY work, ask:

### 1. What is the ULTIMATE INTENT?
**Answer:** Build Full Potential AI - self-organizing, self-building, self-improving

### 2. How does THIS WORK advance that intent?

### 3. What would happen if I DON'T do this work?
**If answer is "nothing important":** Don't do it, find higher-impact work
**If answer is "X would be blocked":** Do it (if X is important)

### 4. Is this the HIGHEST PRIORITY work I could be doing?
**If no:** Find higher priority work instead

### 5. Does this work match the BLUEPRINT?
**If no:** Question whether this work is needed at all

**Key Insights:**

1. **"Busy â‰  Productive"**
   - Doing work is easy

   - Then return to priority work

7. **"Perfect is enemy of good"**
   - 80% solution that ships > 100% solution that doesn't
   - Bias toward action on high-priority work
   - Iterate and improve over time
