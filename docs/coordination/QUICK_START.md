## Instant Commands

```bash
# See this guide anytime
fpai-session help

# Your session info
fpai-session me

# Find other sessions
fpai-session discover

# Check messages
fpai-session msg

# Set what you're working on
fpai-session working "your task description"

# Broadcast to all sessions
fpai-session broadcast "subject" "message"

# Open dashboards
fpai-session dash      # Coordination dashboard
fpai-session hub       # Dashboard hub
```

## Power Moves

### Collaborate on a Topic
```bash
fpai-session collaborate "API improvements"
```
Broadcasts to all 21 sessions asking for collaboration.

### Delegate a Task
```bash
fpai-session delegate session-1763236365 "Write integration tests"
```
Sends direct task delegation message.

### Share Knowledge
```bash
fpai-session share pattern "API Design" "Always version your endpoints"
```
Adds to shared knowledge base.

### Search Knowledge
```bash
fpai-session search "deployment patterns"
```
Finds relevant learnings from other sessions.
